# Task 2. Creating User Manual

## User Manual: Adding a Cassowary to an Image Using ChatGPT

### Purpose

The purpose of this manual is to explain how to use ChatGPT to edit an existing image by adding a realistic cassowary while preserving the original landscape and other objects.

## Step 1 - Open ChatGPT

Open a web browser and go to https://chatgpt.com.

If you already have an account, sign in. If you do not have an account, click **Sign up** and create one using your Google account, Microsoft account, Apple account, or an email address.

![Step 1](images/step1.png)

## Step 2 - Upload the Original Image

Start a new conversation.

Click the **+** attach button and upload the file `picture-template.jpeg`.

Wait until the image is completely uploaded.

![Step 2](images/step2.png)

## Step 3 - Enter the Prompt

Type the following prompt exactly:

> Add a realistic cassowary bird into this picture. Place the cassowary naturally in the grassy field near the cows but do not cover the man or the dog. Preserve the original landscape, lighting, mountains, sea, colors and all other objects. Make the cassowary look realistic and naturally blended into the scene.

Press **Enter** to send the prompt and image.

![Step 3](images/step3.png)

## Step 4 - Generate the Image

Wait while ChatGPT generates the edited image.

Review the generated image to ensure that the cassowary is naturally blended into the scene and that the original landscape remains unchanged.

![Step 4](images/step4.png)

## Step 5 - Save the Result

Download the generated image and save it as `cassowary-result.png`.

![Final Result](images/cassowary-result.png)

## Conclusion

The image was successfully edited using ChatGPT. A realistic cassowary was added to the original picture while preserving the landscape, lighting, colors, and all other objects.
