Civil Engineering - Introduction to AI Final Exam

Academic Program: Civil Engineering

This web application calculates GPA based on the student's transcript. It contains completed, failed, and currently in-progress courses, plus five additional Civil Engineering courses.

Information about GPA i GPA formula translated from Ilia State University regulation:

GPA is calculated by multiplying each course's Grade Point (GP) by the number of credits for that course, adding these values together, and dividing by the total number of credits used in the calculation.

Formula: GPA = Σ(GP × course credits) / Σ(course credits).

GP scale: F = 0-50 points = 0.00; E = 51-60 = 1.00; D = 61-70 = 1.83; C = 71-80 = 2.67; B = 81-90 = 3.50; A = 91-95 = 3.92; A+ = 96-100 = 4.00. For scores between 61 and 95, GP increases by approximately 1/12 for each point.

Hover over or click the information icon to read the English translation of the GPA rule.
